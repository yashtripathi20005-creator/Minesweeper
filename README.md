# Minesweeper Clone

A classic Minesweeper game built with Python and Pygame.

## 🎮 How to Play

- **Left-click** to reveal a cell
- **Right-click** to place/remove a flag
- **R** key to reset the game
- **1, 2, 3** keys to switch difficulty (Easy/Medium/Hard)

## 🚀 Running the Game

```bash
pip install pygame
python main.py
